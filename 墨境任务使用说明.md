# 墨境任务系统 - 用户使用说明

## 功能概览

墨境任务系统现在支持：
- ✅ 查看当日任务（包括未完成和已完成）
- ✅ 查看所有历史完成任务
- ✅ 对已完成的任务进行再次练习
- ✅ 查看每个任务的所有练习记录和评审

## 快速开始

### 1. 今日任务

打开墨境 → 点击"今日任务"

**你会看到：**
- 📊 顶部显示今日进度（已完成/总任务/今日XP）
- 📝 未完成的任务（最多6个）
- ✅ 今天已完成的任务

**操作：**
- 点击任务卡片 → 开始做任务
- 完成后返回，任务变为已完成状态

### 2. 已完成任务库

点击"✅ 已完成"标签

**你会看到：**
- 📚 所有历史完成过的任务
- 🔄 每个任务的练习次数
- ⭐ 每个任务的最高分
- 📅 最后一次练习时间

**操作：**
- 使用筛选器：按任务类型或属性筛选
- 点击"再次练习"：重新练习这个任务
- 点击任务卡片：查看历史记录

### 3. 再次练习

在已完成任务中，点击"再次练习"按钮

**流程：**
1. 确认要再次练习
2. 自动跳转到任务页面
3. 重新输入内容并提交
4. 获得新的评分和反馈

**特点：**
- ✨ 每次练习都是独立的
- ✨ 可以看到自己的进步
- ✨ 每次都能获得XP和属性点

### 4. 查看练习历史

在已完成任务中，点击任务卡片（不是按钮）

**历史记录包含：**
- 📊 总练习次数、完成次数、最高分
- 📝 每次练习的详细信息：
  - 日期时间
  - 得分
  - 字数、耗时
  - 获得的XP
  - 内容预览
- 🔍 点击可展开查看AI详细评审

## 使用技巧

### 💡 提升写作能力
1. 完成一个任务后，查看AI评审
2. 过几天后，对同一任务再次练习
3. 对比两次的分数和评审，看到进步

### 💡 针对性训练
1. 在已完成任务中使用属性筛选
2. 找出某个属性（如"对话力"）的所有任务
3. 重复练习，强化这个能力

### 💡 巩固练习
1. 找出之前分数不高的任务
2. 再次练习，争取更高分数
3. 查看历史，看到自己的进步轨迹

- 🟦 **蓝色徽章** = 墨点任务（5分钟）
- 🟢 **绿色徽章** = 墨线任务（20分钟）
- ✓ **打勾图标** = 已完成
- → **箭头图标** = 未完成

## 常见问题

**Q: 当日任务最多显示几个？**
A: 未完成任务最多6个（3墨点+3墨线），已完成任务全部显示。

**Q: 再次练习会影响之前的记录吗？**
A: 不会。每次练习都是独立的，之前的记录会完整保留。

**Q: 可以查看多久之前的任务？**
A: 所有历史完成的任务都可以查看，没有时间限制。

**Q: 再次练习还能获得XP吗？**
A: 能！每次完成都会获得相应的XP和属性点奖励。

**Q: 如何找到之前练习分数不高的任务？**
A: 在已完成任务列表中，查看"最高分"指标，找出分数较低的任务再次练习。

## 建议使用方式

### 每日练习
- 每天完成2-3个新任务
- 回顾1-2个之前的任务

### 周复习
- 每周查看本周完成的所有任务
- 对分数较低的任务进行二次练习

### 能力提升
- 关注自己薄弱的属性
- 筛选该属性的所有任务
- 反复练习直到掌握
